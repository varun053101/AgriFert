# AI Powered Fertilizer Recommendation Engine

Get AI-powered fertilizer recommendations to grow healthier crops and improve your farm's yield. Smart soil analysis and personalized recommendations for sustainable farming.

## Project info

This project is a React application powered by AI to help farmers make informed decisions about fertilizers.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:

```sh
git clone <YOUR_GIT_URL>
```

2. Navigate to the project directory:

```sh
cd Fertilizer-Recommendation-frontend
```

3. Install dependencies:

```sh
npm install
```

4. Start the development server:

```sh
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
